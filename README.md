A practical migration prototype for moving an Oracle schema and data to Microsoft SQL Server, using SSMA for schema conversion and Python for repeatable data migration and validation.

Implemented:

1. Oracle XE in Docker as source system

2. SQL Server as target

3. SSMA-based schema conversion

4. successful migration of tables

5. preservation of keys and indexes

6. Python-based automated migration workflow

7. validation of migrated row counts / structure
